### Donate crypto to the creator of the tipbot (greenbigfrog#4461)
```
donate [amount] [msg?]
```
## Examples
- `donate 10`: Donates 10
- `donate 10 Thanks for the bot`: Donates 10 as well as sends a the message `"Thanks for the bot"` to greenbigfrog#4461
